gulp-starter
============

Boilerplate for starting a new gulp project

<h3>
	Re-naming
</h3>
<p>
	Remember to rename your gulpfile.js "THEME_NAME" + the path to theme (if you create WP theme).
</p>
<h4>
	Get startet
</h4>
<p>
	Initially you gotta run: <pre>suno npm install</pre>
</p>
<p>
	Then run <pre>gulp</pre>
</p>
