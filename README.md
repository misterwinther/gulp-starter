gulp-starter
============

Boilerplate for starting a new gulp project - This setup is with Bootstrap 3.0

<h3>
	Re-naming
</h3>
<p>
	Remember to rename your gulpfile.js "THEME_NAME" + path
</p>
<h4>
	Get startet
</h4>
<p>
	Initially you gotta run: <pre>sudo npm install</pre>
</p>
<p>
	Then run <pre>gulp</pre>
</p>
